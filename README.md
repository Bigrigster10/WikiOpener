# WikiOpener 📦

**Unbox the world.** WikiOpener is a high-energy, terminal-style case opening simulator that leverages the power of the **MediaWiki Action API** to turn global knowledge into digital loot.

## 🚀 Overview
WikiOpener brings the excitement of competitive case opening to the web. Instead of static skins, you discover everything from rare automotive prototypes and obscure historical artifacts to exotic energy drinks and quantum physics anomalies. Every unboxing is a discovery.

## 🛠️ Features
* **Dynamic Data:** Real-time integration with the Wikipedia API to generate endless loot pools.
* **Terminal Aesthetic:** High-energy, dark-mode UI designed for quick, dopamine-fueled unboxing.
* **Rarity System:** Weighted drop logic based on real-world rarity tiers (Consumer Grade to Exceedingly Rare).
* **Progression:** Inventory system with trade-up contracts, rarity filters, and "StatTrak-style" tracking.
* **Responsive:** Optimized for both mobile flicking and desktop clicking.

## 💻 Tech Stack
* **Frontend:** HTML5, CSS3, JavaScript (ES6+)
* **Integration:** MediaWiki Action API
* **Deployment:** Vercel
* **Storage:** Browser `IndexedDB` & `localStorage`

## ⚙️ How It Works
WikiOpener fetches random article titles, categories, and media from Wikipedia. Each item is assigned a rarity tier and a virtual market value, creating a living economy that mimics top-tier gaming simulators.

## 📦 Run Locally
1. Clone the repository:
   `git clone https://github.com/yourusername/wiki-opener.git`
2. Install dependencies:
   `npm install`
3. Run the development server:
   `npm run dev`

## ⚖️ Disclaimer
This is a game simulator for entertainment purposes only. All items, rarities, and "market values" are entirely virtual and have no real-world cash value. There is no possibility of withdrawing real money.
